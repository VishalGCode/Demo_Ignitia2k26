# 🚀 Ignitia 2K26 – College Fest Website

A modern, responsive, and interactive website developed for **Ignitia 2K26**, the annual technical and cultural fest. The project is built using **HTML5, CSS3, and JavaScript**, featuring a visually engaging interface, smooth animations, and a fully responsive design for all devices.

## 🌐 Live Demo

**Website:** https://2k26ignitia.vercel.app/

> The project showcases the design and user experience for the Ignitia 2K26 event website.

---

## ✨ Features

* 🎨 Modern and attractive UI
* 📱 Fully responsive design
* ⚡ Smooth scrolling and animations
* 🎯 Interactive JavaScript components
* 🖥️ Cross-browser compatibility
* 🚀 Fast and lightweight performance
* 📅 Event-focused landing page
* 💻 Built with pure HTML, CSS, and JavaScript (no frameworks)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
